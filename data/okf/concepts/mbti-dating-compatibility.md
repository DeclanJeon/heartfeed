# Definition

MBTI (Myers-Briggs Type Indicator) compatibility explores how different personality type combinations interact in romantic relationships. Types are categorized by Introversion/Extraversion, Sensing/Intuition, Thinking/Feeling, and Judging/Perceiving.

# Common goals

- Understanding communication style differences between types
- Identifying potential friction points in type combinations
- Using personality awareness to improve relationship dynamics
